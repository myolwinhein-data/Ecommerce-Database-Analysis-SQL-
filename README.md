# Ecommerce-Database-Analysis ( SQL )
## Project Objective
The project aims to transform raw transactional data into actionable business intelligence.

## Business Questions ( KPIs )
- Where do our customers live, and how is our gender distribution?
- Which payment methods are most popular, and which product categories drive the most revenue?
- Who are our "Whale" customers, and how do they rank against each other?
- Which products are at risk of stocking out?
